## React Cats

>This was just a quick experiment making use of the catsaas api created by @kevinbalicot

Demo: https://codesandbox.io/s/wq5wl7vo3k

### Installation
`npm i -save react-cats`

### Usage
* Import the Cat component
`import Cat from 'react-cats'`

* Render it with joy!
```jsx
//.......
function App() {
  return (
    <div className="App">
      <Cat />
    </div>
  );
}
//........
```
